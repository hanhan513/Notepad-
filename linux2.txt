BASH(Bourne Again SHell):			命令别名设定功能： (alias)
									alias lm='ls -af'
									工作控制、前景背景控制： (job control, foreground, background)
									变量的使用与设定：echo, 变量设计规则, unset
									查看环境变量：env export(自定义转环境).查看所有变量：set
									语系变量：lcale,
									发量键盘读取、数组与宣告： read, array, declare
									文件系统及程序的限制关系： ulimit(限制建立文件大小等)
									历史命令：history
									进站画面：/etc/issue
									
									Ctrl + C 终止目前的命令
									Ctrl + D 输入结束 (EOF)，例如邮件结束的时候；
									Ctrl + M 就是 Enter 啦！
									Ctrl + S 暂停屏幕的输出
									Ctrl + Q 恢复屏幕的输出
									Ctrl + U 在提示字符下，将整列命令初除
									Ctrl + Z 『暂停』目前的命令
									
									通配符
									* 代表『0 个b无穷多个』任意字符
									? 代表『一定有一个』任意字符							
									[ ]同样代表『一定有一个在括号内』的字符(非任意字符)。例如 [abcd] 代表『一定有一个字
									符， 可能是 a, b, c, d 这四个任何一个』					
									[ - ]若有减号在中括号内时，代表『在编码顺序内的所有字符』。例如 [0-9] 代表 0 b 9 Y间
									的所有数字，因为数字的诧系编码是连续的！
									[^ ]若中括号内的第一个字符为数符号 (^) ，那表示『反向选择』，例如 [^abc] 代表 一定
									有一个字符，骋是非 a, b, c 的其他字符就接┑囊馑肌
									
									# 批注符号：这个最常被使用在 script 当中，规为说明！在后的数据均不执行
									\ 跳脱符号：将『特殊字符戒通配符』还原成一般字符
									| 管线 (pipe)：分别两个管线命令的界定(后两节介绍)；
									; 连续令下达分别符：连续性命令的界定 (注意！不管线命令圆幌嗤)
									~ 用户的家目录
									$ ㄓ梅⑹前导符：亦即是发量Y前需要加的发量ù值
									& 工作控c (job control)：将令发成背景下工作
									! 逡辑运算意义上的『非』 not 的意思！
									/ 目录符号：路径分别的符号
									>, >> 数据流重导向：输出导向，分别是『ù』不『累加』
									<, << 数据流重导向：输入导向 (这两个留待下节介绍)
									' ' 单引号，不具有发量置换的功能
									" " 具有变量置换的功能！
									` ` 两个『` 』中间为可以先执行的令，亦可使用 $( )
									( ) 在中间为子 shell 的起始不结束
									{ } 在中间为命令区块的组
